2. Allgemeines zur Datenverarbeitung
LifeBuffer ist ein kontextuelles Assistenzsystem. Die App verarbeitet personenbezogene Daten ausschließlich im Rahmen der beschriebenen Funktionen und unter Beachtung der geltenden Datenschutzvorschriften (insbesondere DSGVO).
Die Nutzung der Microsoft-Integration erfolgt freiwillig.
________________________________________
3. Microsoft-Kalenderintegration
3.1 Art der verarbeiteten Daten
Sofern der Nutzer eine Verbindung zu seinem Microsoft-Konto herstellt, verarbeitet LifeBuffer ausschließlich folgende Kalenderdaten:
•	Titel von Kalendereinträgen
•	Start- und Endzeitpunkte
•	optionale Metadaten (z. B. Ort oder Beschreibung), sofern vorhanden
Es erfolgt:
•	kein Zugriff auf E-Mails
•	kein Zugriff auf Kontakte
•	kein Zugriff auf Dateien
•	kein Zugriff auf Chat-Inhalte
________________________________________
3.2 Zweck der Verarbeitung
Die Verarbeitung dient ausschließlich:
•	der Erkennung zeitlicher und thematischer Zusammenhänge
•	der kontextuellen Aufbereitung von Informationen
•	der strukturierten Darstellung relevanter Themen
LifeBuffer erzeugt:
•	keine Aufgaben
•	keine Erinnerungen
•	keine Leistungs- oder Verhaltensprofile
________________________________________
3.3 Art der Verarbeitung
•	ausschließlich lesender Zugriff auf Kalenderdaten
•	keine Änderungen, Ergänzungen oder Löschungen von Kalendereinträgen
•	keine Rückübertragung von Daten an Microsoft
________________________________________
3.4 Rechtsgrundlage
Die Verarbeitung erfolgt auf Grundlage von:
Art. 6 Abs. 1 lit. a DSGVO (Einwilligung).
Die Einwilligung ist freiwillig und jederzeit widerrufbar.
________________________________________
3.5 Widerruf
Die Microsoft-Verbindung kann jederzeit innerhalb der App getrennt werden.
Beim Trennen:
•	werden gespeicherte OAuth-Tokens gelöscht
•	endet jeglicher Zugriff auf Kalenderdaten
________________________________________
4. Speicherdauer
Es erfolgt keine dauerhafte Vollspiegelung des Kalenders.
Daten werden nur insoweit gespeichert, wie es zur Funktion der App erforderlich ist.
________________________________________
5. Weitergabe an Dritte
Es erfolgt keine Weitergabe von Kalenderdaten an Dritte.
Für die Microsoft-Integration gelten zusätzlich die Datenschutzbestimmungen von Microsoft.
________________________________________
6. Rechte der betroffenen Personen
Nutzer haben das Recht auf:
•	Auskunft
•	Berichtigung
•	Löschung
•	Einschränkung der Verarbeitung
•	Datenübertragbarkeit
•	Widerruf einer Einwilligung
Anfragen können an die oben genannte Kontaktadresse gerichtet werden.
________________________________________
7. Sicherheit
Die Verarbeitung erfolgt nach dem Prinzip „Privacy by Design“.
OAuth-Tokens werden serverseitig gespeichert.
Es erfolgt kein direkter Client-Zugriff auf Microsoft APIs.
